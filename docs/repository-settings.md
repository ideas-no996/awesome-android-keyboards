## Social Preview

Suggested social preview image:

- Size: 1280 x 640 px.
- Format: PNG.
- Text: "Awesome Android Keyboards" and "Android IME, Rime, F-Droid, privacy keyboards".
- Visual idea: a restrained Android keyboard grid, a small IME cursor/candidate bar, and subtle labels for Rime, Trime, F-Droid, and privacy.
- Style: clean technical documentation, high contrast, no stock photos, no vendor logos, no exaggerated claims.
- Colors: neutral dark or light background, one Android-green accent, one blue accent for links/metadata.

Set it in GitHub: repository Settings -> General -> Social preview.
